
# algo-Encrypt

Данный проект был создан в учебных целях для изучения алгоритмов шифрования и хэширования.



## Установка

```bash
  git clone https://github.com/versiyaV/algo_encrypt.git
  cd algo_encrypt
  cd sha-256
  make
```
    
## Использование

```bash
./example <argv>
```
